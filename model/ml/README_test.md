# Контрольный объект для проверки модели

## Описание

В директории `ml/` созданы файлы для тестирования модели машинного обучения:

- `test_object.json` - пример входных данных (контрольный объект недвижимости)
- `test_model.py` - скрипт для запуска предсказания модели

## Запуск тестирования

```bash
cd /path/to/project/model
python ml/test_model.py
```

## Формат входных данных

Контрольный объект содержит типичные поля для оценки недвижимости:

- `rooms`: количество комнат (int)
- `area`: общая площадь в м² (float)
- `kitchen_area`: площадь кухни в м² (float)
- `level`: этаж (int)
- `levels`: этажность дома (int)
- `building_type`: тип здания (строка, например "панельный")
- `object_type`: тип объекта ("вторичка" или "новостройка")
- `region`: код региона (строка, например "3" для Москвы)
- `latitude`/`longitude`: координаты (float)
- Дополнительные поля адреса

## Выходные данные

Скрипт выводит результат предсказания в формате JSON, включая:

- Предсказанную цену в рублях
- Цену за м²
- Дельту относительно листинговой цены
- Уровень уверенности
- Топ-факторы влияния
- Объяснение оценки

## Примечания

- Модель использует логарифмическую трансформацию цены
- Применяются рыночные ограничения для реалистичности
- Поддерживаемые регионы ограничены данными обучения
