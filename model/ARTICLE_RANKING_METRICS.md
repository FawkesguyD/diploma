## Расшифровка результатов

Лучшее значение NDCG@10: Статистический метод (1,000).

По метрике NDCG@10 модель CatBoost получила значение 0,982. Это на 0,02 пункта ниже лучшего базового метода (Статистический метод); относительное отличие составляет -1,80%.

По метрике Precision@10 модель CatBoost получила значение 90,0%. Это на 10,00 процентного пункта ниже лучшего базового метода (Статистический метод); относительное отличие составляет -10,00%.

Лучшее значение ProfitCapture@10: Статистический метод (1,000).

По метрике ProfitCapture@10 модель CatBoost получила значение 0,957. Это на 0,04 пункта ниже лучшего базового метода (Статистический метод); относительное отличие составляет -4,26%.

Для инвестора это означает, что в текущей proxy-проверке верхняя часть списка лучше формируется статистическим и эвристическим baselines, а CatBoost не показывает преимущества по основным ranking-метрикам. Такой вывод нельзя переносить на реальную доходность сделок без независимых transaction prices или экспертной оценки рыночной цены.
